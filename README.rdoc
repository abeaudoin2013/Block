== README

Hi, welcome to Block. A rails app designed to help break up writer's block.

Please be sure to run rake db:migrate in the command line. 

Our team has created some prompts that are built in to the repository. 

In order to access them and for the app to function correctly, please run rake db:seed.

Have fun and keep writing!
